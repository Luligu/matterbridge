Quick Installation

Get Matterbridge running in minutes with these simple commands.

NPM Installation

npm install -g matterbridge --omit=dev

On Linux and macOS you may need sudo.

sudo npm install -g matterbridge --omit=dev

After installation start it manually with:

matterbridge

For more installation options, including running as a system service, see the Advanced Configurations section.

Docker Installation

docker run --name matterbridge -v ~/Matterbridge:/root/Matterbridge -v ~/.matterbridge:/root/.matterbridge --network host --restart always -d luligu/matterbridge:latest

The Docker container starts Matterbridge automatically.

Access the Frontend

After starting Matterbridge (via Node.js or Docker) the web frontend is available by default on port 8283.

http://localhost:8283
http://[::1]:8283
http://<your-host-ipv4-address>:8283
http://[<your-host-ipv6-address>]:8283
